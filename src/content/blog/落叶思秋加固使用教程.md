#正文
##1.1 准备工作
首先，你需要准备如下材料：
1.一个脑子
2.一双手
3.一台电脑（手机也可以，需要termux，照着Linux的教程搞就好，手机安卓系统也是基于Linux）
准备完后，点击打开落叶思秋开源地址，转到点击打开releases页面，下载最新版。考虑到GitHub时常打不开，我已经上传好最新版本了，点我不限速下载

下载完成后，先检查电脑有没有Java环境，如果没有Java环境，则无法使用，那么该如何检查呢？
Windows电脑win+r打开运行，输入cmd，打开命令行（macOS和Linux则是Terminal终端），之后分别输入这两个代码：
```
java -version
javac -version  # 检查JDK（开发工具包，必须安装，而非仅JRE）
```
如果输出类似版本号一样的东西，就代表你已经完成了整篇教程里面我个人认为最难的地方了，如果提示找不到命令，不是内部外部指令一类的，就代表你没有Java环境，需要配置。
##1 环境配置
点我打开Adoptium Temurin官网，你用其他的也可以，不过我更推荐这个（下载慢的话开个国外网络，我也懒得上传了）。根据系统下载完成最新版本之后，建议一路确认，啥都不要改，如果要改的话，确保安装路径**无中文，无空格**。
##1.1 环境变量配置
###1.1.1 Windows系统：
右键“此电脑” → “属性” → “高级系统设置” → “环境变量”。
进入环境变量区域后，新建一个“JAVA_HOME”，变量名“JAVA_HOME”，变量值就是JDK安装路径，如C:\Program Files\Java\jdk-11.0.20
新建完成后，编辑“Path”变量，新建一行“%JAVA_HOME%\bin”并上移到顶部，避免冲突，
确定保存
验证：
```
java -version
javac -version
echo %JAVA_HOME%  # 应输出 JDK 路径
```
1.1.2 macOS/Linux系统：
```
# 打开配置文件（zsh 示例）
```

- 在文件末尾添加：
```
# 设置 JAVA_HOME（根据实际安装路径修改）
# macOS（Homebrew 安装）：
export JAVA_HOME=/usr/local/opt/openjdk@11
# macOS（手动安装）：
# export JAVA_HOME=/Library/Java/JavaVirtualMachines/temurin-11.jdk/Contents/Home
# Linux（手动安装）：
# export JAVA_HOME=/usr/local/jdk11
# Linux（包管理器安装）：
# export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
# 将 JDK 加入 PATH
export PATH=$JAVA_HOME/bin:$PATH
```

- 生效配置：
```
# zsh
source ~/.zshrc
# bash
source ~/.bash_profile
```
验证：
```
java -version
javac -version
echo $JAVA_HOME  # 应输出 JDK 路径
2.运行程序
解压刚刚下载的dpt-shell项目，也就是落叶思秋加固项目到任意一个文件夹，然后win+x管理员打开Windows 
```

PowerShell，输入命令：
```
cd 落叶思秋项目加固解压之后的文件夹路径
```

回车运行之后，再输入命令：
```
java -jar dpt.jar -f C:\Users\Administrator\Desktop\你的应用.apk
```

路径可以自由更改，回车运行之后，正常来讲就会开始加固，加固完后，找到落叶思秋项目解压之后的文件夹，里面就会出现已经加固好的应用了。
新人第一次发帖，如果有不对的地方还请大佬指正！由于我没有macOS和Linux系统的电脑，所以环境配置的部分是让ai写的，如果有不对的地方就只能自己研究了，我也不太懂。
